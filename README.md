# Battleship

Welcome to battelship simulator. This is a simulator game, just to play automatic mode.

### Icons Description

- 🛶 -> Ship of 2 positions
- 🛥️ -> Ship of 3 positions
- ⛴️ -> Ship of 4 positions
- 🚢 -> Ship of 5 positions
- 💥 -> Hit a ship
- 💧 -> Water or missing shot
- 🔥 -> Represent a position of a sunken ship
- ' ' -> Empty position

## Reequirements

- Node.js

## Setup

- clone the project
- run `node index.js`
